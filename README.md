# OPC Club — 一人公司俱乐部

> 一个人，就是一家公司。

OPC Club 是一个为独立创业者、自由职业者、独立开发者而生的社群与展示平台。在这里，每一个一人公司都值得被看见、被连结、被支持。

## 这是什么

OPC（One Person Company）指由一个人独立运营的商业实体 —— 他们通常没有固定雇员，但能够独立创造可持续的收入和价值。

OPC Club 的存在，就是要让这些独立创业者被看见：他们的故事、他们的服务、他们的成就，都值得一个舞台。

## 能做什么

- **浏览成员目录**：认识来自技术开发、设计创意、内容创作、营销咨询、教育培训、摄影视频等各行各业的独立创业者
- **搜索与筛选**：按名称、技能、城市或类别快速找到感兴趣的 OPC
- **查看成员详情**：了解每位成员的个人简介、核心技能、提供的服务、业务数据和成就里程碑
- **中英双语**：支持中文与英文界面切换

## 加入条件

只要你在独立运营一家一人公司，有稳定的业务和收入，欢迎申请加入。我们会在 7 个工作日内回复审核结果。

## 加入后的好处

- 获得一个专属展示页面，被其他成员和潜在客户发现
- 加入私密社群，参与定期交流活动
- 获得成员互相推荐和合作的机会
